# 毕业论文管理系统
> 这是一个毕业论文管理系统的前端项目<br>
> 使用VUE框架ElementUI组件库搭建<br>
> 后端项目配合GTManagementSystem-Java使用<br>


## Project setup
```
npm install
```

### Compiles and hot-reloads for developmentS

npm run serve
